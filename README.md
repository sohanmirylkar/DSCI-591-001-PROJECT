# Open Doors International Student Analysis

## Overview

This project analyzes international student trends in the United States using the Open Doors dataset published by the Institute of International Education (IIE).

The objective of this project is to explore patterns in international student enrollment, including country origins, fields of study, and institutional distribution.

## Dataset

Source:
Open Doors International Student Census

The dataset includes multiple tables such as:

- Historical international student enrollment
- New international student enrollment
- Country of origin statistics
- Field of study distribution
- Institutional enrollment statistics

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Exploratory Data Analysis

The project includes the following steps:

1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Visualization of enrollment trends
4. Country-level analysis
5. Institutional inequality analysis using Lorenz curve and Gini coefficient

## Key Findings

- International student enrollment has grown significantly over time.
- A small number of countries account for a large portion of international students.
- International students are concentrated in a small number of universities.
- STEM and business fields dominate international student enrollment.
